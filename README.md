# 🐾 PetCare Hub

A responsive and user-friendly Pet Care website developed using **HTML, CSS, and JavaScript**. The project is designed to provide pet owners with essential information about pet care, adoption, veterinary services, and pet-related resources through a clean and modern user interface.

## 🔗 Live Demo

**Website:** https://srishtipetweb.netlify.app/

## 📖 Overview

PetCare Hub is a front-end web application that focuses on delivering an engaging user experience while showcasing responsive web design principles and interactive JavaScript functionality. The website is optimized for different screen sizes and demonstrates modern UI/UX practices.

---

## ✨ Key Features

- Responsive design for Desktop, Tablet, and Mobile
- Modern and clean user interface
- Interactive navigation menu
- Pet adoption information
- Pet care guidance section
- Veterinary services section
- Pet accessories showcase
- Contact section
- Smooth scrolling experience
- Interactive JavaScript components
- Optimized page layout
- Cross-browser compatibility
- Well-structured and reusable code
